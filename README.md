# Elevation Luxury Editorial Fashion

A modern and stylish luxury fashion editorial website built with **React** and **Vite**.  
Designed to showcase high-end fashion collections and products with smooth navigation and elegant UI.

---

## Features
- Responsive homepage with hero section
- Product pages with images and descriptions
- Smooth animations using **Framer Motion**
- Icon integration with **Lucide React**
- Fast development and build using **Vite**

---

## Tech Stack
- **Frontend:** React, TypeScript, Vite  
- **Animations:** Framer Motion  
- **Icons:** Lucide React  
- **Styling:** CSS / Tailwind (if used)  
- **Version Control:** Git & GitHub
